What is Spray.IO?
=================

  "Spray is an open-source toolkit for building REST/HTTP-based
   integration layers on top of Scala and Akka."

* Fast and lightweight HTTP Server

* Elegant DSL for API Construction

* Support for Servlet 3.0 Containers
   
* Merged into Akka 

* What is Akka?

  "Akka is a toolkit and runtime for building highly concurrent,
   distributed, and fault tolerant event-driven applications on the JVM."
